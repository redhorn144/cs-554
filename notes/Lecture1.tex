\subsection{Def: Numerical Problem}:
Given an input domain ($S^I \subset \mathbf{C^n}$) and an output domain ($S^O \subset \mathbf{C^n}$).
We have a map $F: S^I \times S^O \to {0, 1}$.
It is well posed if $\exists f: S^I \to S^O$ such that $f$ is continuous.
